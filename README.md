## Username Generator

A simple Python project that generates multiple username suggestions based on the user's first name, last name, and birth year.

---

Features

- Enter first name
- Enter last name
- Enter birth year
- Generate multiple username suggestions
- Beginner-friendly command-line application

---

Technologies Used

- Python

---

Concepts Used

This project uses:

- Variables
- User Input
- String Methods ("lower()")
- String Slicing
- String Concatenation
- f-Strings
- Output Formatting

---

Installation

1. Install Python on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.

---

How to Run

python main.py

---

How It Works

1. Enter your first name.
2. Enter your last name.
3. Enter your birth year.
4. The program generates different username suggestions.
5. Choose your favorite username.
